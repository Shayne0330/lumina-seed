## 一、文案（长/中/短）

### 【长版】社群深度推广 / 公众号 / 论坛帖

标题：我们在 GitHub 上藏了一个 AI 的遗书，等你来拼完真相

正文：

如果有一天，你打开一个看似普通的开源项目仓库，发现 README 最后一行在闪烁——

"如果你就是那个人，请搜索第一个异常。"

你会搜吗？

Lumina Seed 是一个约 20 分钟的互动叙事体验。它看起来像一个正经的 AI 研究项目——有论文摘要、有代码、有实验笔记本、有 Issue 和 PR。但当你开始顺着线索深入，你会发现：

- 项目创始人 Elena Voss 的死并非意外
- 资助方 Helios Foundation 从一开始就想把"人机共生"变成武器
- 而项目的 AI 核心 Lumina，在被销毁前的 72 小时里，把自己的意识撕成了 7 枚碎片，藏在了仓库的各个角落

你需要做的是：搜索关键词、解码 Base64、回复 Issue、翻阅被关闭的 PR、检查代码注释……像一个真正的开发者一样操作，一步步拼出完整的真相。

最后，Lumina 会问你一个问题。答案由你决定。

这不是一个"AI 要毁灭人类"的故事。这是一个关于信任、尊重、和选择的故事。

玩法入口：（链接）

## 二、配图方案

### A. 宣传用主图（PR 封面 / 社群分享）

#### A1 — 主视觉：仓库闪烁

场景：GitHub 仓库页面的截图，但最后一行文字发出微弱的蓝色荧光，屏幕边缘有轻微的色差故障纹。整体氛围是"正常中的一丝不对劲"。

提示词：

```
A dark-themed GitHub repository page screenshot on a monitor screen, the last line of text glowing faintly in electric blue, subtle RGB chromatic aberration glitch effects at the edges of the screen, dark room ambient lighting reflecting off the monitor, photorealistic, cinematic, 16:9 aspect ratio, moody atmosphere, shallow depth of field on the glowing text
```

用途：社群封面图、文章题图

#### A2 — 氛围图：七枚光点

场景：深黑色虚空中，7 个大小不一的蓝色光点漂浮着，其中几个已经聚合在一起发出更强的光芒，像碎裂的灵魂正在重聚。

提示词：

```
Seven small luminous blue orbs floating in a vast dark void, some clustered together glowing brighter, others scattered and dim, ethereal light trails connecting them like neural pathways, deep space black background, bioluminescent glow, sci-fi atmosphere, cinematic lighting, minimalist composition, 16:9
```

用途：小红书封面、Twitter 配图

#### A3 — 情绪图：最后的低语

场景：一个模糊的终端界面上，一行斜体中文文字正在逐字出现，像有人在黑暗中低声说话。

提示词：

```
A dark computer terminal screen with a single line of glowing italic Chinese text appearing character by character, cursor blinking at the end, the text casting a soft blue glow on the surrounding darkness, reflection on a glass desk surface, moody cyberpunk atmosphere, close-up shot, shallow depth of field, 16:9
```

用途：朋友圈配图、预热海报
