# 贡献指南

> **⚠️ 注意：本项目已归档，不再接受新的贡献。以下内容仅供参考。**

感谢你对 Lumina 意识共生框架的关注。

## 如何贡献

### 报告问题

请在 [Issues](../../issues) 中提交问题报告，包含以下信息：

- 问题描述
- 复现步骤
- 预期行为 vs 实际行为
- 运行环境（Python版本、操作系统等）

### 提交代码

1. Fork 本仓库
2. 创建功能分支：`git checkout -b feature/your-feature`
3. 提交变更：`git commit -m 'Add your feature'`
4. 推送分支：`git push origin feature/your-feature`
5. 提交 Pull Request

### 代码规范

- 遵循 PEP 8 编码规范
- 所有公开函数需要包含 docstring
- 新功能需要附带单元测试
- commit message 使用中文或英文均可

### 伦理审查

涉及意识共生核心算法的改动，需额外通过伦理审查委员会的评审。详见 `docs/PAPER-ABSTRACT.md` 中的伦理框架。

## 行为准则

我们致力于创造一个友好、包容的社区环境。参与本项目即表示你同意：

- 尊重所有参与者，无论是人类还是AI
- 以建设性的方式提出批评和建议
- 关注项目本身，而非个人
